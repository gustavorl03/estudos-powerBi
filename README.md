## 📚 Conteúdo por nível

### 🟢 Básico
Fundamentos do Power BI para criação de visualizações e relatórios simples.

| Pasta | Conteúdo |
|---|---|
| `01_importacao_dados` | Conexão com fontes de dados (Excel, CSV, banco de dados) |
| `02_tipos_de_graficos` | Gráficos de barras, linhas, pizza, dispersão e mapas |
| `03_filtros_e_segmentacao` | Segmentações, filtros de página e filtros visuais |
| `04_formatacao_visual` | Temas, cores, fontes e layout do relatório |

### 🟡 Intermediário
Modelagem de dados e introdução ao DAX para análises mais robustas.

| Pasta | Conteúdo |
|---|---|
| `01_modelagem_de_dados` | Esquema estrela, tabelas fato e dimensão |
| `02_relacionamentos_tabelas` | Criação e gestão de relacionamentos |
| `03_dax_basico` | CALCULATE, SUM, DIVIDE, RELATED |
| `04_medidas_calculadas` | Criação de medidas para KPIs |
| `05_colunas_calculadas` | Diferença entre medidas e colunas calculadas |

### 🔴 Avançado
DAX avançado, Power Query e storytelling com dados.

| Pasta | Conteúdo |
|---|---|
| `01_dax_avancado` | FILTER, ALL, RANKX, TOPN, inteligência de tempo |
| `02_power_query` | Transformação e limpeza de dados com M |
| `03_kpis_e_indicadores` | Cartões, medidores e indicadores de desempenho |
| `04_storytelling_com_dados` | Narrativa visual, hierarquia e drill-through |

### 🏆 Projetos Práticos
Dashboards completos aplicando os conhecimentos adquiridos.

| Projeto | Descrição |
|---|---|
| `dashboard_vendas` | Análise de desempenho comercial |
| `dashboard_financeiro` | Acompanhamento de indicadores financeiros |

## 📝 Como os arquivos estão organizados

Como arquivos `.pbix` são binários e não podem ser visualizados diretamente no GitHub, cada pasta contém um `notas.md` seguindo este padrão:

```markdown
## Tema: Nome do tema estudado

### O que aprendi
Descrição do que foi estudado nessa aula/prática.

### Recursos utilizados
- Tipo de gráfico / função DAX / transformação usada

### Print do visual criado
![print](print.png)

### Dicas e observações
Alguma dificuldade encontrada ou dica importante.
```

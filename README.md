# 📊 Power BI para Análise de Dados

Este repositório contém meus projetos e estudos utilizando **Power BI** para análise e visualização de dados.

O foco é transformar dados brutos em dashboards interativos que auxiliem na tomada de decisões.

---

## 🎯 Objetivos de Aprendizado

- Importação e transformação de dados
- Modelagem de dados
- Criação de dashboards interativos
- Visualização eficiente de dados
- Análise de métricas e indicadores
